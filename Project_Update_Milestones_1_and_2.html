MILESTONE 2 – AUTHENTICATION & ADMIN DASHBOARD

Objective: Implement secure authentication and create base admin panel layout with vendor management.

Deliverables

  • Admin login with NextAuth (Super Admin & Partner Admin roles)
  • Role-based access control and protected routes
  • Dashboard layout with sidebar and topbar
  • Vendor management (Create, Read, Update, Delete)

Authentication System

NextAuth integration with Supabase for secure authentication. Role-based access control implemented for Super Admin and Partner Admin roles with protected route middleware.

Dashboard Layout

Responsive dashboard with sidebar navigation and topbar. UI upgraded with shadcn/ui components including buttons, cards, tables, dialogs, inputs, labels, badges, and pagination controls.

User Management

Super Admins can create users via modal interface. Automated welcome emails sent with secure password reset links. Default password system implemented for seamless onboarding.

Vendor Management

Full CRUD functionality for vendor management. Super Admins and Partner Admins can add, edit, delete, and view vendors with proper access controls.

Pagination System

Data tables (Users, Coupons, Vendors) feature customizable pagination with 5, 10, 20, 50, or 100 items per page. Preferences persisted in localStorage per page type.

Coupon Statistics

Dashboard displays total, available, and claimed coupon counts at the top of the Coupons page for quick insights.

SOFT DELETE & TRASH FUNCTIONALITY

Enhanced data protection implemented across the system through soft delete and trash functionality for Users, Coupons, and Vendors.

  • Soft Deletion: Items marked as deleted remain in database for 30 days, preventing accidental data loss.
  • Trash Section: Dedicated Super Admin trash interface accessible via sidebar.
  • Restore Capability: Super Admins can view and restore soft-deleted items.
  • Permanent Deletion: Automatic permanent deletion after 30 days, or manual deletion by Super Admin.
  • Audit Trail: System tracks who deleted each item and when.
