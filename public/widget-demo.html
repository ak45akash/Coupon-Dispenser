Programmatic Initialization

Initialize widget using JavaScript API

// Initialize widget programmatically
CouponWidget.init({
  vendorId: 'YOUR_VENDOR_ID',
  userId: 'USER_ID',
  campaignId: 'CAMPAIGN_123',
  theme: 'light',
  containerId: 'my-coupon-widget',
  title: 'Special Offer',
  description: 'Get exclusive discounts'
});
